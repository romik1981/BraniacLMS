class mainapp.views.ApproveTeacherStatus(**kwargs)¶
The get function is used to approve a teacher.

It takes in the request and pk of the user as parameters. It then gets the user object from User model using get_object_or_404 function, which returns a 404 error if no such object exists. Then it sets is_teacher approved attribute of that user to True and saves it in database.

Parameters:

  • self – Represent the instance of the object itself

  • request – Pass the request object to the view

  • pk – Get the user object from the database

Returns:

The user and sets the is_teacher_approved to true
